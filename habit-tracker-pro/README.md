# 📊 Habit Tracker Pro

Habit Tracker Pro is a modern and responsive web application designed to help users build positive habits, track daily progress, and maintain consistency. The application provides an intuitive interface for managing habits, monitoring streaks, tracking completion rates, and visualizing personal productivity.

## 🚀 Features

* Add New Habits
* Complete Daily Habits
* Delete Habits
* Daily Progress Tracking
* Streak Counter System
* Completion Percentage
* Total Habits Counter
* Completed Habits Counter
* Progress Bar Visualization
* Export Habit Data (JSON)
* Local Storage Support
* Dark / Light Mode
* Responsive Design
* Toast Notifications

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 📖 How To Use

1. Enter a habit name.
2. Click "Add Habit" to create a new habit.
3. Mark habits as completed when finished.
4. Track daily progress through the progress bar.
5. Monitor streak counts for consistency.
6. Delete habits when no longer needed.
7. Export habit data for backup purposes.
